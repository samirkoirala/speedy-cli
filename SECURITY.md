# Security Policy

## Supported Versions

The latest release and `main` branch are supported for security updates.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report security issues via:
- Email: samirkoirala13@gmail.com
- Subject: `[SECURITY] speedy-cli vulnerability report`

Please include:
- Affected version/commit
- Reproduction steps / proof of concept
- Impact and suggested remediation

We aim to acknowledge reports within 72 hours and provide a remediation plan.
